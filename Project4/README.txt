Project 4: Traveling Saleman Project

To compile:
	Include the makefile and program file main2.c in the same folder
	To compile simply type 'make' into the command line

To run: 
	To run, type the executable 'program' followed by a valid input file
	with cities and their corresponding x and y coordinates. Example
	run: 

	program tsp_example_1.txt

	The resulting path of this program execution will be outputed to 
	file tsp_example_1.txt.tour
